# Interface: VoiceAssistantAudioSettings

Voice assistant audio settings for configuring audio processing.

## Properties

| Property | Type | Description |
| ------ | ------ | ------ |
| <a id="autogain"></a> `autoGain` | `number` | The automatic gain control setting. |
| <a id="noisesuppressionlevel"></a> `noiseSuppressionLevel` | `number` | The level of noise suppression to apply. |
| <a id="volumemultiplier"></a> `volumeMultiplier` | `number` | The volume multiplier for audio output. |
