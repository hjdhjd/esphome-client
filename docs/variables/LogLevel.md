# Variable: LogLevel

```ts
const LogLevel: {
  DEBUG: 4;
  ERROR: 1;
  INFO: 3;
  NONE: 0;
  VERBOSE: 5;
  VERY_VERBOSE: 6;
  WARN: 2;
};
```

Log levels supported by ESPHome for log subscriptions. These control the verbosity of log messages received from the device.
