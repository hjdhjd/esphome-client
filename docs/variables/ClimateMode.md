# Variable: ClimateMode

```ts
const ClimateMode: {
  AUTO: 6;
  COOL: 2;
  DRY: 5;
  FAN_ONLY: 4;
  HEAT: 3;
  HEAT_COOL: 1;
  OFF: 0;
};
```

Climate modes supported by ESPHome climate entities. These define the primary operating state of HVAC systems.
