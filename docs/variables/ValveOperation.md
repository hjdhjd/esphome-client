# Variable: ValveOperation

```ts
const ValveOperation: {
  IDLE: 0;
  IS_CLOSING: 2;
  IS_OPENING: 1;
};
```

Valve operation states that indicate the current activity of a valve. These represent what the valve is actively doing.

## Type Declaration

| Name | Type | Default value |
| ------ | ------ | ------ |
| <a id="property-idle"></a> `IDLE` | `0` | `0` |
| <a id="property-is_closing"></a> `IS_CLOSING` | `2` | `2` |
| <a id="property-is_opening"></a> `IS_OPENING` | `1` | `1` |
