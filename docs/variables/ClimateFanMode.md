# Variable: ClimateFanMode

```ts
const ClimateFanMode: {
  AUTO: 2;
  DIFFUSE: 8;
  FOCUS: 7;
  HIGH: 5;
  LOW: 3;
  MEDIUM: 4;
  MIDDLE: 6;
  OFF: 1;
  ON: 0;
  QUIET: 9;
};
```

Climate fan modes supported by ESPHome climate entities. These control how the fan operates within the HVAC system.
