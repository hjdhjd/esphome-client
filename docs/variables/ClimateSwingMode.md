# Variable: ClimateSwingMode

```ts
const ClimateSwingMode: {
  BOTH: 1;
  HORIZONTAL: 3;
  OFF: 0;
  VERTICAL: 2;
};
```

Climate swing modes supported by ESPHome climate entities. These control the direction of airflow from the HVAC system.
